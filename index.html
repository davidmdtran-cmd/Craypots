🔥 Firebase Setup

One-time setup — takes about 3 minutes. You only do this once.

  1. Go to console.firebase.google.com
  2. Click Add project → name it (e.g. craypots) → Continue
  3. In your project, click Realtime Database → Create database
  4. Choose Start in test mode → Enable
  5. Click the ⚙️ gear → Project settings → scroll to Your apps
  6. Click </> (Web) → register app → copy the firebaseConfig object
  7. Paste it below (just the object { apiKey: "...", ... })

📋 Rules

  • Start with $0, 1 boat (100% durability), 10 craypots
  • Deploy as many pots as you own (need at least 1 working boat)
  • Boats survive storms but lose 25% durability
  • At 0% durability your fleet cannot sail — repair first!
  • Storm + Deep Water: all deployed pots destroyed
  • Storm + Coastal: pots earn $0 but survive safely
  • Visit the Shop between rounds to buy, repair, or upgrade
⚠️ Sharks now hunt your Golden Craypot in every storm. Defend it here.

🦈 Shark Rules

  • A shark appears in 70% of storms when you own the Golden Craypot
  • Solve the multiplication question in 15 seconds to save it
  • Wrong answer or timeout: shark steals the Golden Craypot
  • Shark Cage charges are consumed one at a time automatically
  • Depth Charge is one-use only per game — use it wisely!
